Create a modern, ultra-stylish, and minimalist e-commerce marketplace website tailored for the Uzbekistan market. The entire user interface must be in the Uzbek language (using proper Latin alphabet like o‘, g‘, sh, ch). The design should look premium, clean, and spacious (similar to a high-end clothing or tech boutique, but simplified to 5 core screens). 

Please implement the following 5 screens/pages using a cohesive and beautiful UI framework (like Tailwind CSS):

1. **Bosh sahifa (Home Page):**
   - Clean header with a logo placeholder, a prominent search bar, and a shopping cart icon.
   - A large, beautiful hero banner showcasing a trendy product or promotion.
   - A horizontal scrolling grid of categories with minimal round icons (e.g., Kiyim-kechak, Elektronika, Goʻzallik, Uy jihozlari).
   - A styled product grid titled "Siz uchun tavsiyalar" with beautiful product cards (large images, rounded corners, soft shadows, clear pricing in 'so'm').

2. **Katalog va Qidiruv (Catalog & Search Page):**
   - A clean layout showing categories and subcategories.
   - High-quality, modern filter buttons (by price, popularity) that look elegant.
   - A grid of search results that updates smoothly.

3. **Mahsulot sahifasi (Product Detail Page):**
   - A large, premium image gallery for the selected product.
   - Clear, bold pricing (e.g., "120 000 soʻm") and a fast shipping badge ("Ertaga yetkaziladi").
   - A prominent, stylish action button: "Savatga qoʻshish".

4. **Savat (Cart Page):**
   - A beautiful list of added items with smooth plus/minus buttons to change quantity.
   - An elegant order summary card showing "Mahsulotlar", "Yetkazib berish (Tekin)", and a bold "Jami" (Total) price.
   - A primary action button: "Buyurtmani rasmiylashtirish".

5. **Profil va Buyurtma (Checkout & Profile Page):**
   - A minimalist, frictionless checkout form asking only for "Telefon raqami" and "Yetkazib berish manzili" (or choosing "Topshirish punkti").
   - A beautiful modal or success screen that appears after clicking "Buyurtma berish" saying: "Rahmat! Buyurtmangiz qabul qilindi ✨".

**Design Requirements:**
- Use a modern font stack (like Inter or Sans-Serif) that renders Uzbek characters perfectly.
- Colors: Use a clean, premium color palette (e.g., crisp white backgrounds, dark elegant text, and a vibrant primary color like deep purple or energetic blue for buttons).
- Ensure the app is fully responsive (looks amazing on both mobile phones and desktop).
- Populate the marketplace with realistic dummy products popular in Uzbekistan (e.g., smartphones, modern traditional clothing, accessories) with prices in Uzbek 'soʻm'.
